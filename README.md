# ga-ops-automation-kit

##GA Ops Automation Kit는 총무팀의 반복적인 운영 업무를 자동화하기 위한 오픈소스 템플릿입니다.

이 프로젝트는 자산 관리, 비품 요청, 회의실 예약, 계약 만료 관리, 업체 연락 내역 정리, 사내 공지 작성 등 다양한 총무 업무를 표준화하고 자동화하는 것을 목표로 합니다.

##주요 기능
비품 요청 목록 자동 정리
자산 관리 데이터 분류
계약 만료일 및 갱신 일정 체크
업체별 연락 및 처리 이력 정리
총무 업무 요청 우선순위 분류
주간 운영 리포트 Markdown 자동 생성
사내 공지 및 안내문 초안 생성용 프롬프트 템플릿 제공
CSV/JSON 기반 입력 데이터 지원
사용 예시

총무 담당자가 아래와 같은 데이터를 입력하면:

비품 구매 요청 목록
사무실 시설 점검 요청
계약 만료 예정 목록
회의실 예약 및 변경 요청
임직원 문의 내역

프로그램은 이를 자동으로 정리하여 다음과 같은 결과물을 생성합니다.

처리 우선순위 목록
담당자별 업무 분류표
주간 총무 운영 리포트
공지문 초안
계약 갱신 체크리스트
프로젝트 목적

총무 업무는 조직 운영에 필수적이지만, 반복적이고 수작업 비중이 높은 경우가 많습니다.
이 프로젝트는 총무팀이 단순 반복 업무에 쓰는 시간을 줄이고, 더 중요한 운영 개선 업무에 집중할 수 있도록 돕기 위해 시작되었습니다.

특히 중소기업, 스타트업, 비영리단체, 학교, 커뮤니티 운영팀 등 별도의 업무 시스템을 도입하기 어려운 조직도 쉽게 활용할 수 있도록 오픈소스로 공개합니다.
